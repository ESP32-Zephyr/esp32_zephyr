Wifi

File: wifi.h

Description: WiFi interface abstraction.

Defines function pointer types for WiFi operations and the WiFi interface struct containing those function pointers. Provides an interface function to get the WiFi interface instance.

@struct wifi_iface_t WiFi interface containing function pointers for WiFi operations.
