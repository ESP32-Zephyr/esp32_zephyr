Server

File: server.h

Description: Server abstraction interface.

Defines the server type, server protocol enumeration, and interface functions for creating, destroying, and starting a server.

server_t *server_new(server_proto_t type, int port, _Bool register_service)
[source]

Create a new server instance.

Allocates and initializes a new server of the specified protocol type, listening on the given port. Optionally registers the service if requested.

Parameters:

  • type – The server protocol type (UDP or TCP).

  • port – The port number on which the server will listen.

  • register_service – Set to true to register the service, false otherwise.

Returns:

Pointer to the newly created server instance, or NULL on failure.

void server_destroy(server_t **self_p)
[source]

Destroy a server instance.

Deallocates and cleans up the server instance pointed to by self_p. After this call, the pointer will be set to NULL.

Parameters:

  • self_p – Pointer to a pointer to the server instance to destroy.

void server_start(void *p1, void *p2, void *p3)
[source]

Start the server.

Starts the server. The parameters p1, p2, p3 are generic pointers whose meaning depends on the implementation and server type.

Parameters:

  • p1 – Pointer to context or first argument.

  • p2 – Pointer to context or second argument.

  • p3 – Pointer to context or third argument.
